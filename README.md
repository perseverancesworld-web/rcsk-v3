# RCSK v3: Recursive Cognitive Swarm Kernel

A self-reflective, multi-agent cognitive operating system kernel with governed evolution, persistent causal memory, and distributed swarm capabilities.

## Quick Start

```bash
pip install -e .
python -m runtime.run_local
```

## Core Features
- Cognitive Swarm: Planner, Critic, Executor, Shadow, Dream agents
- Memory: Causal graph with salience and replay
- Evolution: Hot-swappable modules with sandbox evaluation
- Governance: Drift detection and approval gates
- Observability: Full event logging and time-travel replay
- Swarm: Multi-node distributed execution

See `docs/architecture.md` for details.
